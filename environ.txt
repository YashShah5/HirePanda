flowchart TD
    A[Start: Run Script] --> B[Load environment variables from .env]
    B --> C[Authenticate to source GitHub (SOURCE_TOKEN + ORG)]
    B --> D[Authenticate to destination GitHub (DESTINATION_TOKEN + ORG)]

    C --> E[Fetch SOURCE repo]
    D --> F[Fetch DESTINATION repo]

    E --> G[Fetch tags from source repo]
    F --> H[Fetch tags from destination repo]

    G --> I[Compare tag names]
    H --> I

    I --> J{Are there missing tags?}
    J -- Yes --> K[Write missing tags to CSV]
    J -- No --> L[Log: All tags present]

    K --> M[End]
    L --> M[End]
